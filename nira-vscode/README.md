# Nira VS Code Extension

Syntax highlighting and language support for the Nira programming language.

## Installation

To install this extension manually:

1. Copy the `nira-vscode` folder to your VS Code extensions directory:
   - **macOS/Linux:** `~/.vscode/extensions/`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\`

2. Restart VS Code.

Alternatively, run this command in your terminal:

```bash
mkdir -p ~/.vscode/extensions/nira-vscode
cp -r nira-vscode/* ~/.vscode/extensions/nira-vscode/
```

## Features

- Syntax highlighting for keywords, types, functions, and built-ins.
- Support for `.nr` files.
- Basic auto-indentation and bracket matching.
